# Claude Design 프롬프트 — 꾸미기 탭 게임 감성 강화

**사용법**: 전체 블록을 Claude Design 새 대화에 복사. `index.html` + `admin.html` + `DESIGN_SYSTEM.md` 첨부.

---

저는 고연승T 수능 영어 학원 매니저 hyun 이에요. 학생 일차 성적표 안에 **오리 꾸미기 탭**을 운영합니다. 현재 기본 구현은 됐는데, "게임처럼 또 열고 싶다"가 약해요. **재방문률 상승이 핵심 목표**입니다.

## 범위
Detail Card 의 **꾸미기 (DRESS UP) 탭 내부만** 봅니다. 공유 카드·Detail 다른 3탭·관리자 뷰는 범위 밖.

## 현재 구조
- 상단 프리뷰 큰 오리 (5슬롯 실시간 조합)
- 슬롯 선택 칩: 모자 / 상의 / 발 / 탈 것 / 악세서리
- 인벤토리 그리드: 현재 슬롯 14개 아이템 (획득=컬러 미리보기, 미획득=흐리게+LOCK)
- 상태 요약 블록: 장착 세트·획득 총량
- 데이터: 70종 × 5티어(N 15 / R 20 / E 18 / L 12 / M 5)

## 지켜야 할 것
첨부 `DESIGN_SYSTEM.md` 의 모든 규칙. 특히:
- 티어 시각 공식 (N/R/E/L/M) — Apex Legends 5단계 색상 코딩 레퍼런스
- 오리 본체 좌표 (head r=6.8) 그대로
- 학업 성능 영향 암시 **금지** — cosmetic only 고지는 눈에 띄게 유지
- 가챠·랜덤 드롭 시각 요소 금지

## 지금보다 나아지길 바라는 것
1. **슬롯 칩 전환이 게임 장비창 같게** — 지금은 단순 탭 느낌
2. **인벤토리 그리드가 '수집 만족감'을 주게** — 획득/미획득 비율이 눈에 한번에
3. **프리뷰 오리가 '살아있는 느낌'** — 미세 bobbing, 선택한 아이템이 등장하는 트랜지션
4. **미획득 아이템도 미리보기 명확** — 지금은 filter:saturate(0.82)+opacity 0.62 로 했는데 더 좋은 방식 있으면 제안
5. **L/M 티어가 "언젠가 가져갈 것" 으로 매력적으로** — 잠김 상태에서도 유혹적
6. **슬롯별 진도 표시가 '도감 진행률' 느낌** — 6/14 → 5/14 → 컬렉션 스타일 게이지

## 레퍼런스 (참고)
- Animal Crossing: New Horizons 카탈로그 → "있는 것 + 없는 것" 차분히 공존
- Fortnite Battle Pass 아이템 트리 → 시즌감
- Honkai Star Rail 보유 무기 화면 → 희귀도 시각 즉시
- Pokemon GO 도감 → 성장·수집 서사
- Club Penguin 옷장 → 캐주얼하고 쾌적한 장비창

## 미획득 카드 처리 제안
현재: filter saturate(0.82) brightness(0.88) opacity(0.62) + 하단 어두운 scrim + LOCK 뱃지
검토 가능 방향:
- (a) 현재 유지 + 미션 문구 가독성 더 높이기
- (b) 컬러풀 상태 유지 + LOCK 뱃지만 강조
- (c) 블러(backdrop-filter: blur(4px)) + gold halo 테두리
- (d) 실루엣 윤곽만 뚜렷하고 안은 비움 (ACNH 도감 식)

## 산출물
- 꾸미기 탭 전체의 **3안 (A/B/C)** — 레이아웃·시각 강도가 각각 다른 세 버전
- 각 안마다 "어떤 학생 타입에 먹힐지" 한 줄 (예: A=성실파, B=자랑파, C=수집파)
- 마이크로 인터랙션 스펙 3개 (트랜지션 타이밍·easing)
- 인벤토리 카드 1매 상세 (획득·equipped·잠김 3상태)

## 제약
- CSS-only 우선. JS 는 기존 state 로직만 사용, 새 JS 추가 최소화
- 외부 아이콘·이미지 라이브러리 금지 (인라인 SVG 만)
- 실학생 데이터 없음 — 더미 미션 문구 그대로
- 모바일 1열 폴백 유지

먼저 꾸미기 탭의 **약점 3가지** + 제가 못 본 **재방문 유도 레버 2가지** 를 주시고 이어서 A/B/C 안으로 가주세요.
